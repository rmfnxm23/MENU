# MENU

가상의 온라인 메뉴 페이지입니다.
</br>
메뉴를 확인하고, 각 메뉴의 가격과 설명을 보며, 이미지를 클릭해 더 자세한 정보를 얻을 수 있습니다.

</br>
</br>

## 프로젝트 목적 💪

"HTML과 CSS를 활용하여 웹 페이지를 제작하고, 웹 디자인과 레이아웃 구성에 대한 이해를 깊게 하기 위해 이 프로젝트를 시작하였습니다."

</br>
</br>
